# MQTT Manage Frontend (Vue.js)

Vue.js frontend for managing Mosquitto MQTT server via backend API.

## Usage
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start the development server:
   ```bash
   npm run serve
   ```

> **Note:**
> - The frontend expects the backend to be running on the same host and port 4000 (or adjust API URL as needed).
> - Features: status, start/stop, user management, log viewing.
